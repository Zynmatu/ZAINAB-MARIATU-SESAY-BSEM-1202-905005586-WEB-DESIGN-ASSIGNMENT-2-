# SL Library

SL Library is a responsive educational website that provides free access to:

- WASSCE Past Papers
- BECE Past Papers
- NPSE Past Papers
- Educational Books

The platform was created to support students, teachers, and parents in Sierra Leone by improving access to learning resources.

## Built With

- HTML5
- CSS3

## Goal

To promote equal access to quality education through digital learning resources.
